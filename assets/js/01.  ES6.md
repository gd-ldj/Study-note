
1. includes 方法
+ 判断数组是否有某值 `Array.includes('x');`
+  判断字符串中是否包含某个字符串
  ```js
  let str = 'woaini';  
  str.includes('ai'); //true
  ```

2. 不要声明之后又给对象添加新属性，推荐先默认null：
```js
// low
const a = {};
a.x = 3;

// good
const a = { x: null };
a.x = 3;
```
如果一定非要加请使用Object.assign
```js
const a = {};
Object.assign(a, { x: 3 });
```
